# TUBES_GAME_KELOMPOK_GIRLS

## Deskripsi Proyek

Project ini merupakan game 2D yang dikembangkan menggunakan Godot Engine sebagai tugas besar mata kuliah. Game mengimplementasikan berbagai fitur seperti sistem player, enemy, item, HUD, serta mekanisme permainan berbasis aksi.
